
# EXERCI

Exerci is a ringwormGO Gen-X web browser, made in C# (for terminal version) & Java (for GUI version). 

## Version -> 0.0.2

## Required software
* Mono (Terminal version, can be compiled with .NET too)
* Java (GUI version)

## How to run (C#)
```
$ make
$ mono main.exe
```

## How to run (Java)
```
$ java main.java
```
